Download Data

how to export your data from bit.io and save it as a file

suggestion

Download Tables

if you have data uploaded on bit.io and saved in tables which you'd like to upload, you can download it and save it as a local file. on your repo page click download repo on the top right

You can download one table or a subset of tables by checking the boxes next to their name. Clicking on the "Entire Repo" checkbox will download all of the tables in the repo.

Click on "CSV" next to File Format to select which export format you want for this data.

if the total size of the exported data is relatively small, the file will directly download after a few seconds of processing. make sure bit.io has permission from your web browser to save file downloads
f the total size of the table or tables is larger, bit.io will send you an email with a download link for each downloaded table.

Download Query Results

if you'd like to download the results of a SQL query, click the 'download' button on the top right of the data preview window

The query results will download as a CSV file after a few seconds of processing. Make sure bit.io has permission from your web browser to save file downloads.

Supported Export Formats

Currently, bit.io supports exporting data as a CSV, JSON, or Excel (XLSX) file.
